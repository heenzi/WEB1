Tag list


1. strong : 폰트 굵게
2. u : 밑줄
3. h1 to h6 : 제목(heading), 볼드체 + 폰트 크기 1~6
4. br : 줄바꿈 (닫기태그 없음)
5. p : 단락


6. img : 사진 파일
7. src= : 속성. 이미지 파일 경로 > If you wanna know more attributes : ATTRIBUTES
width, height : 속성. 이미지 가로세로 설정

8. 리스트 태그 : ol (ordered list) , ul (unoredered list) + li


9. title : 웹에서 탭에 뜨는 페이지 제목 설정
10. head, body, html : 본문 설명(제목칸), 본문란, 전체 묶는 태그
11. meta chraset : 한글 읽을 수 있게 웹 읽기소스 바꾸는 것


12. a : 하이퍼링크
13. href= : 속성. 하이퍼링크 레퍼런스. 링크 달 주소
14. title= : 속성. 링크 미리보기 제목창
15. target= : 하이퍼링크를 어디서 열지 설정. VALUES
